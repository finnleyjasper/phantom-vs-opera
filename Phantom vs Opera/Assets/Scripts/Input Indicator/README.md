## Manual Spawn (Runtime)
- **Ctrl + indicator key** → spawns a normal input.
- **Shift + indicator key** → spawns a special input.

## Console Logs
- **Start:** shows indicator count, input count, and perfect window.
- **Manual spawn:** logs which key and time the input was placed.
- **PERFECT! (green)** / **FAIL! (red)** — printed on every hit/miss with the key and timestamp.
